ICS: Chapter 2 Bits,Data Types and Operations

Bits, Data Types and Operations

1. Bits and Bytes

  • bit: only 0/1

  • byte: 1 byte = 8bits

2 Data Type

2.1 Integer

2.1.1 Unsigned Integer

n bits can represent 2^n different values, from 0 to 2^n - 1.

2.1.2 Signed Integer

  • signed-magnitude(原码)：n bits, 1 bit for sign, n-1 bits for magnitude
    for example, 0110=6, 1110=-6

  • 1’s complement(反码)：n bits, 1 bit for sign,其余按位取反
    for example, 0110=6, 1001=-6

  • 2’s complement(补码)：n bits, 1 bit for sign,其余按位取反再加1
    for example, 0110=6, 1010=-6
    n位补码所能表示范围为$-2^{n-1}$到$2^{n-1}-1$

    2.2 Floating Point

    IEEE 754标准：

  • 单精度：32位，1位符号位，8位指数位，23位尾数位

  • 双精度：64位，1位符号位，11位指数位，52位尾数位

  • 单精度浮点数范围：$1.18 \times 10^{-38}$到$3.4 \times 10^{38}$，双精度：$5.0 \times 10^{-324}$到$1.8 \times 10^{308}$

位数 符号位(S) 指数位(exp) 尾数位(frac)
32 1 8 23
64 1 11 52

单精度浮点数转换方式：
$$
N = (-1)^S \times 2^{exp-127} \times (1 + frac)
$$
双精度浮点数转换方式：
$$
N = (-1)^S \times 2^{exp-1023} \times (1 + frac)
$$

注意：当指数位全为1时，表示无穷大或NaN（Not a Number）。具体来看，若尾数位全为0，则表示正无穷大或负无穷大；若尾数位不全为0，则表示NaN（非数值）。

2.3 Logical Variables

与：$A·B$ $\Rightarrow$ $A$ AND $B$
或：$A+B$ $\Rightarrow$ $A$ OR $B$
非：$ \bar{A}$ $\Rightarrow$ NOT $A$
异或：$A⊕B$ $\Rightarrow$ $A$ XOR $B$
同或：$A⊙B$ $\Rightarrow$ $A$ XNOR $B$

助教说了，按照模电的形式，即前者写没问题，但是后者也要知道，考试可不会告诉你（）
